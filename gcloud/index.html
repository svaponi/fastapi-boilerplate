Google Cloud Integration

This page contains information on how to configure the application to use resources on Google Cloud. Use-cases:

  • Install private dependencies published on Artifact Registry.
  • Publish docker image on Artifact Registry.
  • Deploy to Google Kubernetes Engine (GKE).
  • Deploy to Cloud Run.

Install Google Cloud CLI

  • Install it from the GCloud Installation page.
  • Run in terminal:
gcloud init
  • You will be asked to sign via Google Sign On.
  • Choose your project from the menu.

Add Google Cloud implementation of keyring to Poetry

  1. Run in terminal:
poetry self add keyrings.google-artifactregistry-auth
